# UHUB

## [ Objetivos🎯 ]
 Esse projeto está sendo criado com o objetivo de ser apresentavo ao final da matéria de Extensão do Curso de Análise e Desenvolvimento de Sistemas.
 
## [ Proposta👨‍💻 ]
 O projeto Uhub tem a proposta de ser um site para centralizar e fazer com diferentes estudades de cursos superiores das faculdades brasileiras possam trocar conhecimento de forma fácil e eficiente, criando entre os usuários a sensação de se estar em uma bilioteca trocando conhecimentos com outras pessoas e tirando suas próprias dúvidas.
 
## [ Funcionabilidade⚙️ ]
O Uhub será um site que utiliza do email acadêmico para verificar a ligação do usuário com uma instituição de ensino superior, permitindo ou não a criação de conta na plataforma. Durante a utilização será disponibilizado uma interface onde o usuário podera encontrar um painel com os principais fóruns discutidos e pessoas onde poderam ter um chat.
